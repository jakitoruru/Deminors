Deminors

Deminors adalah suatu web game yang dibuat sebagai sarana untuk refleksi user dan sebagai wadah inovasi game web kepada game developer lainnya. Game web ini berbasis game online dengan single player. Kita mengambil tema musik karena musik memiliki kekuatan yang luar biasa dalam menginspirasi, menyatukan, dan menghibur. Musik menciptakan ikatan emosional yang kuat, menghadirkan berbagai warna dan nuansa ke dalam pengalaman manusia. Dengan memilih tema musik, kami berharap untuk menyampaikan pesan-pesan kami dengan lebih mendalam dan merangsang imajinasi serta perasaan pengguna melalui medium yang universal dan kuat ini.

Game 1

Game pertama bernama rhythm game. Yang menarik dari game ini adalah konsep bermain yang menantang kecepatan user, dengan cara menoccokan tempo nada dengan waktu yang ditentukan.

Game 2

Game kedua kami berjudul NotNot. Game ini dapat membuat nada-nada dengan hanya menekan tuts lalu dapat di Play semua nada secara keseluruhan.
